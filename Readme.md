# MathGlow ✨

A modern and beautifully designed UP Board Mathematics handbook built using React + Vite.
MathGlow provides a smooth app-like experience with offline support, smart formula search, bookmarks, recent history, dark mode, and premium responsive UI specially optimized for Android devices.

Built for students who want mathematics to feel cleaner, faster, and more interactive.

---

## ✨ Features

* 📘 UP Board Class 9 & 10 Mathematics formulas
* 🔍 Instant formula & chapter search
* 🌙 Dark / Light mode support
* 💖 Bookmark favorite formulas
* 🕘 Recently viewed formulas
* 📱 Android-first responsive design
* ⚡ Fast performance powered by Vite
* 📐 KaTeX mathematical rendering
* 📦 Offline PWA support
* 🎨 Soft glassmorphism + neumorphic UI
* 🧩 Component-based React architecture
* 🚀 Installable as an app on mobile devices

---

## 🛠️ Tech Stack

* React 19
* Vite
* KaTeX
* Lucide React
* CSS3
* JavaScript (ES Modules)
* PWA

# 📥 Installation

## 🖥️ Kali Linux Installation

### 1. Update packages

```bash id="9t38ks"
sudo apt update && sudo apt upgrade -y
```

### 2. Install Node.js & npm

```bash id="wbpk7v"
sudo apt install nodejs npm -y
```

### 3. Install Git

```bash id="27r5es"
sudo apt install git -y
```

### 4. Clone repository

```bash id="8m6ucx"
git clone https://github.com/Jn-Corporation/Maths-Glow.git
```

### 5. Open project folder

```bash id="nd1ck4"
cd Maths-Glow
```

### 6. Install dependencies

```bash id="e2d4r0"
npm install --legacy-peer-deps
```

### 7. Start development server

```bash id="0t2rj4"
npm run dev
```
